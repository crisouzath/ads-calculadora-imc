# ⚖️ Calculadora de IMC

## 📖 Sobre o Projeto

Este projeto consiste em uma **Calculadora de IMC (Índice de Massa Corporal)** desenvolvida com **HTML, CSS e JavaScript**.

A aplicação permite que o usuário informe seu peso e altura para calcular automaticamente o IMC, exibindo o resultado e sua respectiva classificação de acordo com os padrões utilizados na área da saúde.

O desenvolvimento foi realizado a partir da replicação de um protótipo criado no **Figma**, seguindo as orientações apresentadas durante a aula.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Figma

---

## ⚙️ Funcionalidades

* Cálculo automático do IMC.
* Exibição do resultado na tela.
* Classificação do IMC de acordo com o valor calculado.
* Validação dos campos de entrada.
* Interface inspirada em um protótipo do Figma.
* Layout responsivo para diferentes tamanhos de tela.

---

## 🧠 Conceitos Praticados

Durante o desenvolvimento deste projeto foram aplicados diversos conceitos importantes, tais como:

* Manipulação do DOM.
* Captura de eventos com JavaScript.
* Utilização de `EventListener`.
* Atualização dinâmica de informações na tela.
* Criação e chamada de funções.
* Estruturação semântica com HTML.
* Estilização com CSS.
* Desenvolvimento responsivo utilizando `@media screen`.
* Adaptação de layouts para diferentes resoluções.
* Replicação de interfaces a partir de protótipos no Figma.
